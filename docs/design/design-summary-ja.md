## 適用デザイン原則

- **ミニマリズム:** ダークネイビー + ホワイト + 単一のゴールドアクセント — 競合する色なし
- **技術的信頼性:** 具体的な数値（0.05 µm、ISO 17025、98.7%）がエンジニアリング権威を表現
- **視覚的ヒエラルキー:** 見出しにInter Tight 700、本文にInter 400 — 全セクション統一
- **ゴールド = アクションシグナル:** CTAと主要指標、セクションラベルのみに使用 — 装飾的ノイズなし
- **セクションリズム:** ライト（`#f5f7fa`）→ ダーク（`#003087`）→ ライト → ダーク（`#001840`）で交互コントラスト
